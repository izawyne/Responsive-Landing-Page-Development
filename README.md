# Responsive Landing Page Development
Developed and deployed a responsive landing page using HTML, CSS, and JavaScript as part of a front-end web development project.

Key features:

- Responsive layout design for desktop and mobile devices
- Interactive user interface using JavaScript
- Structured styling and component organization with CSS
- Version control using Git and GitHub
- Deployment and hosting using Netlify

Technologies:
HTML, CSS, JavaScript, Git, GitHub, Netlify
